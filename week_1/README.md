# 🚀 Go Backend Mastery

## 📅 Hafta 1 — Go DNA: Tilning O'zagi
Bu haftaning asosiy maqsadi — Go-ning falsafasini tushunish va **"Go-cha"** fikrlashni (Idiomatic Go) shakllantirishdir.


### 📚 O'rganish Mavzulari

#### 1. Type System & Composition
* **Inheritance vs Embedding:** Go-da merosxo'rlik (inheritance) yo'qligini, uning o'rniga `struct embedding` qanday ishlashini o'rganish.
* **Interface Satisfaction:** Interfeyslarning `implicit` (duck typing) tabiati — struktura interfeysni "so'zsiz" qanday amalga oshirishi.

#### 2. Interfaces — Go-ning Eng Kuchli Qismi
* **Standard Library Interfaces:** `io.Reader`, `io.Writer` va `error` interfeyslarini chuqur tahlil qilish.
* **Decoupling:** Interfeyslar yordamida komponentlarni bir-biridan qanday mustaqil qilish.

#### 3. Pointers & Receivers
* **Pointer vs Value Receiver:** Qachon pointer, qachon qiymat (value) ishlatish kerak? 
* **Memory Efficiency:** Nusxa olish va xotira manzili bilan ishlashning unumdorlikka ta'siri.

#### 4. Error Handling (Go Style)
* **Idiomatic Errors:** `if err != nil` oqimi.
* **Advanced Error Handling:** * `errors.Is` va `errors.As` funksiyalari.
    * `fmt.Errorf` va `%w` yordamida error wrapping.
    * **Sentinel errors** vs **Custom error types**.

#### 5. Slices & Maps: Ichki Tuzilish
* **Slice Header:** Xotirada `ptr`, `len` va `cap` qanday saqlanishi.
* **Map Mechanics:** Xesh-kolliyalarni boshqarish va xotira taqsimoti.
